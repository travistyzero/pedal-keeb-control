# Pedal Keeb Control
Control my ZSA Moonlander keyboard using a USB pedal controller.
When the pedal is pressed down, it will switch to the mouse layer.
When the pedal is raised, it will switch back to the previous layer.

This works with an iKKEGOL FS2017U1IR.
